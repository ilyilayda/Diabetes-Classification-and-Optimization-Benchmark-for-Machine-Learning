# Diabetes-Classification-and-Optimization-Benchmark-for-Machine-Learning
Diabetes prediction project using Logistic Regression, SVM, and Neural Networks. Includes data preprocessing, EDA, optimizer benchmarking (SGD, Momentum, Nesterov, AdaGrad, RMSProp, Adam, AdamW), regularization, cross-validation, model comparison, and Kaggle evaluation for performance and generalization analysis.
